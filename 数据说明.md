# 📊 数据范围说明

## ✅ 支持的股票

本系统使用 **AkShare** 数据源，支持：

### 中国 A 股
- 上海证券交易所（60xxxx）
- 深圳证券交易所（00xxxx、30xxxx）

**示例：**
- 贵州茅台：600519
- 平安银行：000001
- 招商银行：600036

## ❌ 不支持的股票

- **美股**（如 NVDA 英伟达、AAPL 苹果、TSLA 特斯拉）
- **港股**（如 00700 腾讯、09988 阿里巴巴）
- **其他国际市场**

## 🎯 为什么会出现"空谈"

### 问题示例

你查询：**"分析英伟达的走势"**

AI 回答：
```
技术分析：
尽管无法直接获取详细的历史数据...  ← ❌ 这是空谈！
我们可以查看最近的股价表现来推断...  ← ❌ 没有真实数据
```

### 原因

1. **英伟达是美股** - AkShare 不支持
2. **工具调用失败** - 获取不到数据
3. **旧版 Prompt 不够严格** - AI 会"编造"内容

## 🔧 技术说明

### AkShare 数据范围

AkShare 是一个开源的财经数据接口库，主要提供：
- ✅ 中国 A 股实时/历史数据
- ✅ 股票新闻
- ✅ 技术指标
- ✅ 行业数据
- ❌ 美股数据（不支持）
- ❌ 港股数据（部分支持，本系统未启用）

### 工具调用流程

1. 用户查询 → AI 解析股票代码
2. AI 调用工具（get_stock_history 等）
3. AkShare 获取数据
4. 返回真实数据 → AI 分析

**如果第3步失败：**
- ❌ 旧版：AI 会编造内容
- ✅ 新版：AI 会明确告知无法获取

## 💡 使用建议

### 正确的查询方式

1. **指定 A 股股票：**
   ```
   分析贵州茅台（600519）的走势
   ```

2. **包含股票代码：**
   ```
   获取平安银行（000001）的新闻
   ```

3. **行业对比（A 股）：**
   ```
   对比宁德时代和比亚迪的技术指标
   ```

### 错误的查询方式

1. **查询美股（无数据）：**
   ```
   分析特斯拉的走势  ← 会被拒绝
   ```

2. **不指定代码（可能识别错误）：**
   ```
   分析腾讯  ← 可能会找 A 股的腾讯相关股票
   ```
